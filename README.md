# Vue Socks Shop (Cart + Reviews)

A small Vue 3 project that simulates a basic product page for socks. Users can switch between color variants, see stock status, add items to a cart, remove items, view pricing, and submit reviews.

## Features
- **Product variants** (green/blue) with images
- **Stock status** (In Stock / Out of Stock)
- **Price displayed next to variant swatches**
- **Cart**
  - Add to cart
  - Remove from cart
  - Cart item count + total cost display
- **Shipping rule**
  - Free shipping when cart total is **$15+**
  - Otherwise shows a paid shipping price
- **Reviews**
  - Add a review using a form
  - Display submitted reviews in a list


Getting Started
1) Install dependencies
npm install
2) Run the dev server
npm run dev
3) Build for production
npm run build
